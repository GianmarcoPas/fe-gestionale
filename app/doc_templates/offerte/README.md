## Template Offerte (.docx)

Metti qui i 4 file `.docx` base con questi **nomi esatti**:

- `off_old.docx`
- `off_old_amm.docx`
- `off_iper.docx`
- `off_iper_amm.docx`

### Placeholder supportati

Nel testo del documento (anche in header/footer) usa questi placeholder:

- `[Cliente]`
- `[Indirizzo]`
- `[Civico]`
- `[Cap]`
- `[Comune]`
- `[Prov]`
- `[P. IVA]`
- `[data emissione]` (es. `18 gennaio 2026`)
- `[importo caricamento]`
- `[importo revisione]`

### Tabella beni (punto 5)

Nella riga “template” della tabella inserisci questi placeholder:

- `[bene 1]`
- `[valore bene 1]`
- `[importo offerta 1]`

Se il lavoro ha più beni, quella riga verrà duplicata automaticamente e compilata per ogni bene.

Nota: i placeholder della tabella sono gestiti anche se scritti con maiuscole (es. `[Bene 1]`).
